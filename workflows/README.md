# Workflows

Each file in this directory is a Markdown SOP defining a workflow.

A workflow should include:
- **Objective** — what it accomplishes
- **Required inputs** — what data/credentials are needed
- **Tools used** — which scripts from `tools/` it calls
- **Steps** — the sequence of actions
- **Expected outputs** — what gets produced and where it goes
- **Edge cases** — how to handle failures or unusual situations
